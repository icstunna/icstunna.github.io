The difference between Relative, Absolute and Fixed Positioning

January 3, 2015

A relative position is a posture that repositions an element in relation to its default position. For example if an element is on the upper left hand corner of an object, and you set the position to relative then give it a position attribute; depending on the attribute the element will move in relation to its default position.
