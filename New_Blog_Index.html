What is Rails?

Written by Mikhail Delos Trinos on February 13, 2015

In 2004 David Heinemeier Hansson developed Ruby on Rails or Rails, while working on "Basecamp" a project management tool by 37signals. He released Rails to be open sourced in 2005,and in 2006, Apple announced that it would ship Rails as part of its Leopard operating system. Rails is an open source web application framework that is written in the programming language Ruby. Rails uses best practice software paradigms such as Model View Controller (MVC), Active Record Pattern (ARP) and Convention over Configuration (COC).

PA MVC is a software architecture that implements user interfaces. The MVC allows the user to use an application in 3 componenets. The user uses a controller in order to make changes to the model / program. Once the input from the user is receive through the controller, the controller then manipulates the model / program with the parameters set by the user. The model / program then takes the information from the controller and updates its state meaning that when there have been changes made it complies to those changes and forwards those changes to the view. The view requests information from the model / program in order to display an output for the user. This whole cycle then loops back to the user.
